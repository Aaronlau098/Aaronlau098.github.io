Vscode快捷键

1. 一般快捷键

快捷键 功能
Ctrl + Shift +P or F1 显示命令面板
Ctrl + P 快速打开，转到文件…
Ctrl + Shift + N 新建窗口/实例
Ctrl + Shift + W 关闭窗口/实例
Ctrl + , 用户（工作台）设置
Ctrl + ` 显示终端
Ctrl + Shift + ` 创建新终端
Ctrl + K Ctrl + S 快捷键设置

2.基础编辑

快捷键 功能
Ctrl + C 复制文本
Ctrl + V 粘贴文本
Ctrl + X 剪切文本
Alt + $\uparrow$ / $\downarrow$ 上下移动行
Shift + Alt + $\uparrow$ /$\downarrow$ 上下复制行
Ctrl + Shift + K 删除行
Ctrl + Enter 下面插入行
Ctrl + Shift + Enter 上面插入行
Ctrl + shift + \ 括号匹配
Ctrl + ] / [ 代码缩进
Home / End 转到行首/行尾
Ctrl + Home / End 转到文首/文尾
Ctrl + $\uparrow$ / $\downarrow$ 行上下滚动
Alt + PgUp / PgDn 页面上下滚动
Ctrl + Shift + ] / [ 展开/折叠区域
Ctrl + K Ctrl + ] /[ 展开/折叠所有子区域
Ctrl + K Ctrl + J /0 展开/折叠所有区域
Ctrl + K Ctrl + C 行注释
Ctrl + K Ctrl + U 删除行注释
Ctrl + / 注释
Shift + Alt + A 切换块注释
Alt + Z 切换自动换行

3.导航

快捷键 功能
Ctrl + T 显示所有符号
Ctrl + G 转到行…
Ctrl + P 转到文件…
Ctrl + Shift + O 转到符号…
Ctrl + Shift + M 显示问题面板
F8 转到下一个错误或警告
Shift + F8 转到上一个错误或警告
Ctrl + Shift +Tab 浏览编辑器组的历史记录
Alt + $\leftarrow$/$\rightarrow$ 后退/前进

4. 搜索和替换

快捷键 功能
Ctrl + F 查找
Ctrl + H 替换
F3 查找下一个
Shift + F3 查找上一个
Alt + Enter 选择所有当前出现的匹配项
Ctrl +D 将选择作为查找匹配项
Ctrl + K Ctrl + D 将上一个选择作为查找匹配项
Alt + C/R/W 查找区分大小写/正则表达式/全字匹配

5. 多光标和选择

快捷键 功能
Alt + Click 插入游标
Ctrl + Alt +$\uparrow$/$\downarrow$ 在上方/下方插入光标
Ctrl + U 撤销上一个光标操作
Shift + Alt + I 在所选的每一行末尾插入光标
Ctrl + L 选择当前行
Ctrl + Shift + L 选择所有当前选择
Ctrl + F2 选择所有出现的当前单词
Shift + Alt + $\leftarrow$/$\rightarrow$ 收缩选择/扩展选择
Shift + Alt + (Drag Mouse) 列框选择
Ctrl + Shift + Alt + (Arrow Key) 列框选择
Ctrl + Shift + Alt + PgUp / PgDown 上下列框选择页面

6. 丰富的语言编辑

快捷键 功能
Ctrl + Space 触发建议
Ctrl + Shift + Space 触发参数建议
Shift + Alt +F 格式化文件
Ctrl + K Ctrl + F 格式化所选代码
F12 转到定义
Alt + F12 查看定义
Ctrl + K F12 侧面打开定义
Ctrl + . 快速解决
Shift + F12 显示参考
F2 重命名符号
Ctrl + K Ctrl + X 修建行末空格
Ctrl + K M 变更语言

7. 编辑管理

快捷键 功能
Ctrl + F4 / Ctrl + W 关闭编辑器
Ctrl + K F 关闭资料夹
Ctrl + \ 分割编辑器
Ctrl + 1/2/3 专注于第一、第二、第三编辑组
Ctrl + K Ctrl + $\leftarrow$/$\rightarrow$ 移动活动的编辑器组

8. 文件管理

快捷键 功能
Ctrl + N 新建文件
Ctrl + O 打开文件
Ctrl + S 保存文件
Ctrl + Shift + S 文件另存为
Ctrl + K S 保存所有文件
Ctrl + F4 关闭文件
Ctrl + K Ctrl + W 关闭所有文件
Ctrl + Shift + T 重新打开关闭的编辑器
Ctrl + K Enter 保持预览模式编辑器处于打开状态
Ctrl + Tab 打开下一个
Ctrl + Shift + Tab 打开上一个
Ctrl + K P 活动文件的复制路径
Ctrl + K R 在资源管理器中显示活动文件
Ctrl + K O 在新窗口/实例中显示活动文件

9. 显示

快捷键 功能
F11 切换全屏
Shift + Alt + 0 切换编辑器布局
Ctrl + =/- 放大/缩小
Ctrl + B 切换边栏可见性
Ctrl + Shift + E 显示文件管理器
Ctrl + Shift + F 显示搜索界面
Ctrl + Shift + G 显示源代码管理界面
Ctrl + Shift + D 显示调试界面
Ctrl + Shift + X 显示扩展界面
Ctrl + Shift + H 替换文件
Ctrl + Shift + J 切换搜索详细信息
Ctrl + Shift + U 显示输出面板
Ctrl + Shift + V 打开Markdown预览
Ctrl + K V 在侧边栏打开Markdown预览
Ctrl + K Z 禅宗模式（esc esc退出）

10. 调试

快捷键 功能
F9 切换断点
F5 开始/继续
Shift +F5 停止
F11 /Shift + F11 进出断点
F10 跳过
Ctrl + K Ctrl + I 显示悬停
